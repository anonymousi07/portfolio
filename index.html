Movie Recommendation System

January 2025

Built and deployed a Movie Recommender System using Python, Streamlit, and machine learning. The system applies content-based filtering with a cosine similarity matrix to suggest movies based on plot, genres, and features. It integrates with the TMDB API to fetch real-time movie data and posters, while combining a user-friendly Streamlit frontend with Python logic on the backend. The project is deployed on Render for easy accessibility and scalability.
